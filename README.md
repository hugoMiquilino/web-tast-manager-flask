# Web Task Manager com Flask

## 📌 Descrição
Aplicação web simples desenvolvida com Flask para gerenciar tarefas, baseada no conceito de CRUD (Create, Read, Update, Delete).

O projeto foi pensado para demonstrar, de forma didática, a interação entre usuário e aplicação web utilizando Python.

## 🚀 Funcionalidades
- Lista tarefas
- Adicionar tarefas
- Marca tarefas como concluídas
- Remover tarefas
- Validação de entradas
- Tratamento de erros para evitar falhas na aplicação

## 🛠️ Tecnologias
- Python
- Flask
- HTML
- CSS básico

## ▶️ Como executar?

1. Clone o repositório ou baixe os arquivos

2. Instale as dependências:
```bash
pip install -r requirements.txt
python app.py
```
3. Acesse no navegador:
```http://127.0.0.1:5000```
