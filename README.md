# Web Task Manager com Flask

## Descrição
Aplicação web simples para gerenciar tarefas, desenvolvida com Flask.

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Remover tarefas

## Tecnologias
- Python
- Flask

## Como executar?

pip install -r requirements.txt
python app.py

Abra o navegador:
http://127.0.0.1:5000
